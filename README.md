# Продвинутое хранилище для игр
Это хранилище решает многие проблемы vk storage, одна из проблем 4096 символов сохранить можно, а с этим хранилищем очень много.  
[bridge](bridge) - Сейчас имеется только версия которая совместима с vk bridge, но в дальнейшем может появиться ещё одна, где кода меньше.
